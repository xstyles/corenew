== Description ==

**WPC AJAX Add to Cart for WooCommerce** is a highly effective plugin for helping online stores cut down the site’s loading time, improve the user experience, and increase sales. This plugin allows products to be added to the cart immediately without reloading the whole site to see the changes. It’s super easy to use, just click Install then Activate it and enjoy yourself.

= Advanced shopping experience =

Using WPC AJAX Add to Cart not only improves the buyers’ experience but also makes the shopping session of your customers more fluent and comfortable with:

- Less idle time of waiting for the site to load
- Stay on track with the item being viewed
- Enjoy hassle-free site navigation

= Widely compatible with WPC plugins =

WPC AJAX Add to Cart for WooCommerce can work with:

- Simple and variable products
- [WPC Product Bundles for WooCommerce](https://wordpress.org/plugins/woo-product-bundle/)
- [WPC Composite Products for WooCommerce](https://wordpress.org/plugins/wpc-composite-products/)
- [WPC Frequently Bought Together for WooCommerce](https://wordpress.org/plugins/woo-bought-together/)
- [WPC Grouped Products for WooCommerce](https://wordpress.org/plugins/wpc-grouped-product/)
- [WPC Force Sells for WooCommerce](https://wordpress.org/plugins/wpc-force-sells/)
- [WPC Smart Quick View for WooCommerce](https://wordpress.org/plugins/woo-smart-quick-view/)
- Most WordPress themes and WooCommerce plugins

Especially, it’s highly recommended to use WPC AJAX Add to Cart with [WPC Fly Cart for WooCommerce](https://wordpress.org/plugins/woo-fly-cart/) plugin to combine the forces of two tools with greatly beautiful effects and user experience.

== Installation ==

1. Please make sure that you installed WooCommerce
2. Go to plugins in your dashboard and select "Add New"
3. Search for "WPC AJAX Add to Cart", Install & Activate it
4. Done!
